Description

This project is an example of an online bank. The program begins with the menu being displayed which allows the user to login, create a new user, or exit. If the user wants to create a new user, then the following information is obtained: First name, last name, username, password, and email. If the user wants to log in, then the user will enter their username and password. Once the username and password is validated, the main menu will appear. The main menu consists of the following options.

  1. Create an account: allows the user to create an account (up to 3)
  2. Add a user to an account: allows the user to add another user to an account. (max 4 users per account)
  3. View balance: shows the balances of all the user's accounts
  4. Withdraw: allows the user to withdraw an amount from an account if they have the funds
  5. Deposit: allows the user to deposit an amount into an account
  6. Transfer: allows the user to transfer an amount from one account to another
  7. Close an account: allows the user to close an account if they have one
  8. Log out: allows the user to log out
