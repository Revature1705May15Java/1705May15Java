Menus

[Short description of menus section]

Initial Menu

The initial menu welcomes the user and allows them to log in, become a member of the bank, or exit the program.

Login Screen

If the user selects the log in option, they are prompted for an email. If they enter an email that is not in the database, they are taken back to the initial menu. Otherwise, they are prompted for their password.

If the user enters an incorrect password three times, the program will terminate. This behavior is used to simulate a user being locked out of their account for too many failed login attempts in a short amount of time.

A successful login will take the user to the user menu.

New Member Screen

Users that are not members of the bank can easily sign up. They need only provide a unique email address, their name, and a password. New users are added to the "Users" table of the database.

After successfully becoming a member of the bank, new users will be taken to the user menu.

Exit Program

Users can also choose to exit the program. Upon choosing this option, all open resources are closed and the user is presented a friendly message.

User Menu

This menu is available to users who have successfully logged in and to new bank members.

Create New Account

This option allows the user to create a new account. Once a new account is created, the user is taken to the account actions menu.

View Account Details

This menu lists all of the user's accounts, and prompts them to choose one. User's are taken to the account actions menu for the account that they have chosen.

Account Actions Menu

This menu displays the chosen account's information and allows users to perform a number of actions on the account.

SQL Stored Procedures

[Short description of stored procedures used in this project]

close_account(a_id IN NUMBER)

Given an account id number, a_id, this procedure will close an existing account.

CREATE OR REPLACE PROCEDURE close_account(a_id IN NUMBER)
	IS
	BEGIN
		UPDATE accounts
		SET close_date = sysdate
		WHERE account_id = a_id;
	END close_account;

create_new_user(fname IN VARCHAR2, lname IN VARCHAR2, pw IN VARCHAR2, email IN VARCHAR2)

This procedure will create a new user with the given first name (fname), last name (lname), password (pw), and email address (email)

CREATE OR REPLACE PROCEDURE create_new_user (fname IN VARCHAR2, lname IN VARCHAR2, pw IN VARCHAR2, email IN VARCHAR2)
	IS
	BEGIN
		INSERT INTO users (first_name, last_name, password, email)
		VALUES(fname, lname, pw, email);
		END create_new_user;
