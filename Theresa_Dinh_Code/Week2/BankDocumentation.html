Documentation of the Banking assignment is here.

Some of the functions included in this assignment allow a usesr to: create a new account; open and close bank accounts; withdraw, deposit, and transfer money, and check the balance of a bank account.

The addUser method is implemented in the service layer, where it takes the information needed to create a new user account and passes it on to the DAO so it can write the data to the database.
In this method, its paramters are Strings containing the information for the new User. We create a temporary User object to store a User's information as we check the database to ensure that the User doesn't already exists. After calling the DAO to read the database records and make sure there doesn't exist one already with the same username/email. If the addition of a new User is written to the database, then the method returns the new User, otherwise it returns a null object.
