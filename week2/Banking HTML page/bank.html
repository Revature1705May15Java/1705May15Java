Mark Worth Bank Project

Variable Layout and Table creation:

These tables were created to hold the user and account information in Oracle SQL

We used the user_account join table to hold the account information for certain users and the user information for certain accounts as it represents an n:n relationship

Code implementation of statements through jdbc

Simple statements are useful for run-once code so they are particularly useful for fetching lookup table data such as the account type table which holds the name related to certain account types, saving some queries

Prepared statements are generally useful for most queries and are less susceptible to SQL injection than regular statements. These were used for the majority of database queries and updates.

The callable statement is particularly useful in the case of complete events which we need to ensure happen on the database side, such as account creation, as we don't want an account which doesn't have a user or some such problem to occur.

Functionality of the program

Users can log into their account or create a new account,

View all of their available accounts,

and withdraw,deposit, or close an account.
